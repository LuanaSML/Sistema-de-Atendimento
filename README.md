# Sistema de Atendimento

> Sistema de Atendimento para registrar e gerenciar chamados.


## Sobre o Projeto

> Sistema para controle de atendimento Sistemas de “tickets” com ferramentas para apoio à gestão do atendimento ao usuário.


## Funcionalidades
- Cadastro de atendimentos
- Visualização de detalhes do chamado
- Controle de status 
- Autenticação básica (com senha se for funcionario)
- painel com 5 ultimas senhas chamadas
- relatórios (quantidade de atendimentos, senhas, tipos, data, hora, etc.)

## Tecnologias
- **Backend:** NodeJS
- **Frontend:** React
